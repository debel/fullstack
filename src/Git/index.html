How does git work?

  • Working directory
  • Index
  • Repository
  • Remotes

Basic local commands:


    $ git init #create a new repository in the current dir
    $ git add #add files to the index
    $ git commit #create a new commit
    $ git status #show the status of the working dir vs the index
    $ git checkout #get specific versions of files
    $ git branch #manipulate branches

More local commands:


    $ git log
    $ git reset
    $ git merge
    $ git rebase

Remote commands:


    $ git clone
    $ git fetch
    $ git push
